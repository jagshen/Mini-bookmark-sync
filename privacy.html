Privacy Policy

极简书签同步（Chrome 扩展）

一、生效日期

本隐私政策自 2026 年 4 月 29 日 起生效。我们保留随时更新本政策的权利，更新后的政策将在本页面发布。

二、开发者信息

扩展名称：极简书签同步

扩展功能：通过 WebDAV 协议在用户的自建/自有云存储服务之间同步 Chrome 浏览器书签数据。

本扩展为个人开发的开源工具，不隶属于任何公司或商业组织。本扩展本身不运营任何服务器、不收集任何用户数据至第三方。

三、数据收集概览

简而言之：本扩展不向任何第三方服务器发送您的数据。所有数据仅存储在您本地浏览器和您自行配置的 WebDAV 云存储中。

数据类型 是否收集 说明
书签数据（标题、URL、文件夹结构） 用户主动操作 仅在用户手动点击"上传"/"下载"/"合并"或启用自动同步时，由用户指定的 WebDAV 地址进行传输
WebDAV 配置（地址、用户名、密码） 仅本地存储 使用 chrome.storage.local 加密存储在浏览器本地，永远不会上传至除您指定 WebDAV 服务之外的任何地方
设备标识符 仅本地生成 本地随机生成的字符串，用于区分不同设备的数据来源，仅存在于本地和用户自己的 WebDAV 文件中
浏览历史 / 网页内容 不收集 本扩展不会记录、读取或上传您的浏览历史与网页内容
个人信息（姓名、邮箱、IP 等） 不收集 本扩展不具备任何用户身份识别或追踪能力
使用统计 / 分析数据 不收集 本扩展不含任何第三方分析 SDK 或遥测代码

四、Chrome 权限使用说明

本扩展申请了以下 Chrome 权限，每项权限的具体用途如下：

  • bookmarks（书签） — 读写 Chrome 书签栏和其他书签文件夹中的书签数据，这是本扩展的核心功能。
  • storage（存储） — 使用 chrome.storage.local 在浏览器本地保存 WebDAV 连接配置（地址、用户名、密码）、同步设置及上次同步时间。所有数据均存储在本地浏览器中，不会上传至第三方。
  • alarms（定时器） — 用于实现用户配置的自动定时同步功能（如每隔 N 分钟自动执行一次书签同步）。
  • host_permissions（网站访问权限） — 仅在用户填写并保存 WebDAV 地址后，动态请求对该特定域名的访问权限，用于通过 HTTP/HTTPS 协议与用户自有的 WebDAV 服务器通信。不会预申请任何网站的访问权限。

五、数据传输方式

  • 书签数据以 JSON 格式通过 HTTPS（推荐）或 HTTP 协议传输到用户自行配置的 WebDAV 服务器。
  • 传输过程使用 Basic Auth 认证，凭据（用户名 + 密码）由用户提供并存储于本地浏览器。
  • 本扩展的开发者无法访问、查看或拦截您 WebDAV 服务器上的任何数据。
  • 数据的最终存储位置完全取决于用户选择的 WebDAV 服务提供商（如坚果云、群晖 NAS、Nextcloud 等用户自管服务），与本扩展无关。

六、数据安全措施

  • WebDAV 密码等敏感信息使用 chrome.storage.local API 存储于浏览器本地，遵循 Chrome 浏览器的沙盒安全机制。
  • 本扩展声明了严格的内容安全策略（Content Security Policy），禁止执行任何外部脚本。
  • 本扩展不包含任何远程代码加载、追踪脚本或第三方 SDK。
  • 网站访问权限采用动态申请机制（optional_host_permissions），仅在用户明确授权后才访问指定域名。

七、第三方服务声明

本扩展本身：

  • 不嵌入任何广告 SDK
  • 不嵌入任何统计分析工具（如 Google Analytics、百度统计等）
  • 不嵌入任何社交分享组件
  • 不连接任何开发者控制的服务器

⚠️ 注意：当您使用本扩展将书签同步到 WebDAV 服务器时，实际的数据托管方是您选择的云存储服务商（如坚果云、Nextcloud 等）。这些第三方服务的数据处理政策请参阅各服务商自身的隐私条款，与本扩展无关。

八、儿童隐私保护

本扩展不面向 13 周岁以下儿童设计，不故意收集任何儿童的个人信息。由于本扩展本身不收集任何用户数据，因此不存在针对儿童的特别风险。

九、用户权利

  • 导出/删除数据：您可以随时在扩展的配置页清除所有本地配置；也可以通过 Chrome 书签管理器（chrome://bookmarks）管理您的全部书签数据。
  • 卸载扩展：卸载本扩展后，所有通过 chrome.storage.local 存储的本地数据将被自动清除。
  • 停止同步：您可以随时关闭自动同步功能或在扩展中移除 WebDAV 配置。
  • 云端数据：存储在您 WebDAV 服务器上的书签文件可随时通过文件管理器直接删除，无需经过本扩展。
